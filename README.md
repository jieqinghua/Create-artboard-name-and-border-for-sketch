# Create-artboard-name-and-border-for-sketch

A sketch plugin that creates name laber and border for all artboards in the page.

画交互稿必备。一键生成画板标题和边框。

之前网上找的插件新版sketch不能用了，所以借助ChatGPT自己重新写了一个。
